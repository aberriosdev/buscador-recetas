# Buscador de Recetas – Gourmet Go

## Sprint 1 – Prototipo Visual
Prototipo visual del Buscador de Recetas.  
Maquetación estática realizada con Bootstrap 5.

## Sprint 2 – Funcionalidad y Conexión a API
Aplicación web para la búsqueda de recetas por ingrediente, utilizando la API TheMealDB.  
Implementa asincronía con fetch y async/await, y renderizado dinámico sin recargar la página.

## Tecnologías
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6+)
- Fetch API
